# TuniData Compass

**AI-Driven Education, Skills & Career Intelligence Platform for Tunisian Students**  

## Project Overview
TuniData Compass is a Business Intelligence and Artificial Intelligence platform designed to analyze the Tunisian job market, university curricula, and student skill profiles.  
The platform delivers actionable insights, skill gap analysis, future skill predictions, and personalized career recommendations for students.

The solution combines ETL pipelines, a data warehouse, machine learning models, and BI dashboards to support students, universities, and recruiters in data-driven decision-making.

## Objectives
- Analyze Tunisian job market trends and skill demand
- Identify skill gaps between academic curricula and industry needs
- Provide personalized career recommendations for students
- Predict future job and skill trends using machine learning
- Deliver interactive dashboards for insights and reporting

## Scope
### Included
- Job market data collection (scraping)
- Data cleaning, transformation, and normalization
- NLP-based skill extraction
- Data warehouse design
- Machine learning models (recommendation, forecasting)
- BI dashboards
- Web interface
- Technical documentation

### Not Included
- Full mobile application development
- Handling sensitive personal student data
- Manual labeling of real student data

## Key Features
- Job demand and skill trend analysis
- Skill gap detection between universities and industry
- Career recommendation engine with Job Fit Score
- Personalized skill development roadmap
- Interactive BI dashboards

## Technical Stack
- **ETL & Data Processing:** Python, Pandas, NumPy, BeautifulSoup, Selenium
- **Data Storage:** Data Lake (CSV/JSON/Parquet), MongoDB Data Warehouse
- **Machine Learning:** NLP (spaCy / BERT), Recommendation (cosine similarity), Forecasting (Prophet / LSTM)
- **Business Intelligence:** Power BI
- **Application Layer:** Streamlit or Flask

## Deliverables
- Raw and cleaned datasets
- ETL pipelines and scripts
- Data warehouse schema
- Machine learning models
- Power BI dashboards
- Web application
- Final technical documentation and PFE report

## Project Type
Academic Final Year Project (PFE)
